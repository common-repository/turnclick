== Description ==
TurnClick's WordPress plugin allows TurnClick users to automatically install the embed code on their WordPress websites.

If you don't have an account yet, <a href="https://www.turnclick.com">sign up for free.</a>

== Installation ==
1. From your WordPress dashboard, select "Plugins" in the left sidebar.
2. Add a new plugin
3. Search for "TurnClick"
4. Install "TurnClick Embed Code"
5. Once installed, go Settings â†’ TurnClick Settings
6. Enter your TurnClick username (your email address) and click "Save Settings".
